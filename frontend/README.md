# React Chat UI

ChatGPT-like interface for RAG MVP.

## Features

- ✅ Real-time chat interface
- ✅ Source citations
- ✅ Confidence scores
- ✅ Mode indicators
- ✅ Performance metrics
- ✅ Error handling

## Prerequisites

- Node.js 16+
- npm or yarn

## Setup

```bash
cd frontend
npm install
```

## Configuration

Create `.env` file:

```
REACT_APP_API_URL=http://localhost:8080/api
```

## Run

```bash
npm start
```

Opens at http://localhost:3000

## Build for Production

```bash
npm run build
```

## Features

- **Chat Interface**: Clean, modern UI
- **Source Display**: Expandable source citations
- **Confidence**: Shows answer confidence score
- **Mode Detection**: Shows detected query mode
- **Timing**: Shows retrieval and generation times
- **Error Handling**: Graceful error messages
